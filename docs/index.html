Features

Everything you need,
nothing you don't

High-quality PDF output

Uses Chrome's native print engine for pixel-perfect rendering — code blocks, math, images, all preserved exactly as they appear in chat.

Select specific messages

Enter select mode to cherry-pick exactly which messages make it into the PDF. Keep the useful parts, skip the noise.

Full export control

A4, A5, or Letter. Portrait or landscape. Custom filename, margin, and scale — all saved between sessions.

How it works

Three clicks to a PDF

01
Install the extension

One click from the Chrome Web Store, Firefox Add-ons, or load unpacked from the GitHub repo.

02
Open any DeepSeek chat

The PaperSeek toolbar appears automatically at the bottom-right of every conversation.

03
Choose messages (optional)

Click Select to handpick which messages to export, or skip this step to export the full conversation.

04
Export PDF

Hit Export PDF. The file downloads instantly — no upload, no server, no waiting. Your chat stays yours.
