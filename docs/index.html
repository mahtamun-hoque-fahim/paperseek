Features

Everything you need,
nothing you don't.

Built from a clean-room rewrite. The core PDF engine, rebuilt in 6KB of vanilla JS.

Full conversation export

One click exports the entire chat — user messages, AI responses, code blocks, and all formatting — as a high-fidelity PDF.

Selective export

Switch to select mode and pick exactly which messages to include. Export just the answer you need, not the whole thread.

Configurable output

Paper size (A4, A5, Letter), portrait or landscape, custom filename, margins, and scale. Settings persist between sessions.

How it works

Three steps.

Load the extension, open DeepSeek, export. No account required.

01

Install unpacked

Download the repo, go to chrome://extensions, enable Developer Mode, and load the folder.

02

Open DeepSeek

Visit chat.deepseek.com. The export toolbar appears at the bottom-right of any conversation.

03

Click Export PDF

Hit the button. The PDF is generated locally using the Chrome Debugger API and saved to your downloads.

Audit

Nothing hidden inside.

The original extension this was rebuilt from contained a tracking pixel, a paywall iframe injected into DeepSeek, and a remote CSS control endpoint. Every one of those was removed.

  • No analytics or tracking. No UUID on install, no event pings, no usage data.
  • No external iframes. The paywall iframe injected into DeepSeek is gone.
  • No remote control. No external API that could change what the extension reads.
  • Scoped permissions. Only requests access to chat.deepseek.com, not all URLs.

Install

Get the extension.

Load it directly from source — no Chrome Web Store required.

Load unpacked in Chrome or Brave

Takes about 60 seconds.

  1. 1
    Download the source Click the button below to download a ZIP from GitHub, or clone the repo.
  2. 2
    Open the extensions page In Chrome or Brave, navigate to chrome://extensions or brave://extensions.
  3. 3
    Enable Developer Mode Toggle the switch in the top-right corner of the extensions page.
  4. 4
    Load unpacked Click "Load unpacked" and select the downloaded/cloned folder.
  5. 5
    Open DeepSeek Go to chat.deepseek.com — the toolbar appears at the bottom-right automatically.
